# Cayley-Hamilton theorem

Every square matrix $\mathbf{A}$ satisfies its own characteristic equation,

$$\mathbf{A}^n + a_{n-1} \mathbf{A}^{n-1} + \dots + a_1 \mathbf{A} + a_0 \mathbf{I} = \mathbf{0} \ .$$
